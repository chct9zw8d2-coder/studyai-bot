# Telegram StudyAI Bot (Text + Images) + Stars (999⭐) + бесплатные лимиты

Функции:
- Текст + картинки через Pollinations.AI (без ключей)
- Доступ на 30 дней за 999 Stars
- PostgreSQL (Railway)
- Бесплатно без подписки:
  - 5 текстовых запросов в день
  - 3 картинки в день

> Время "дня" считается по UTC.

Деплой на Railway — см. README в прошлом архиве (шаги те же).
